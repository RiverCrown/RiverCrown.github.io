《Celeste》

发表于 2018-09-22

Celeste 是一款优秀的平台跳跃类游戏，它获得了 IGN 的满分评价。

我买这款游戏的动机就是因为 IGN 给出了满分的评价。在经过数十个小时的受苦后，我能坦言这款游戏确实配得上这个分数。

剧情

这种平台跳跃游戏居然强行扯上了剧情，不过剧情也很简单，就是主角战胜自己内心的套路故事，然后将这个战胜的过程具象为登山的过程。

虽然整个剧情都很简单，但是如果用心去体会剧情，再结合游戏的画面风格和音乐，相信有相似经历（经历人生低谷）的人们还是会受到治愈。

音乐

Celeste 的音乐很耐听，毕竟作为一个有难度的平台跳跃游戏，玩家最常做的事情就是死亡。其次每一关卡的音乐风格都很契合剧情走向和画面风格（这应该是所有正常游戏都应该做到的）。

难度

平台跳跃类游戏的难度往往是玩家们津津乐道的部分，每当一款有难度的平台跳跃游戏出来，微博下总是充满了手残党们的哀嚎声。

Celeste 作为一款硬核平台跳跃游戏，已经算是十分照顾新手了。Celeste 首先提供了一个表关卡，换句话说就是一周目，手残党体验剧情专用。其次考虑到只寻求体验游戏过程的玩家，Celeste更是提供了帮助模式。所以手残党完全不用担心自己买了一个无法通关的游戏。

还有很重要的一点是 Celeste 把死亡惩罚降到了最低，每次死亡后只是从当前关卡重新来过，且载入速度极快。极大程度上降低了玩家死亡后的挫败感，提升了玩家的游玩体验。

对于进阶玩家来说，Celeste 绝对是时间杀手。在通完 A 面后，玩家可以继续收集全部的草莓，且每个关卡除了通关时的红心外，还有一个蓝心可以收集。做完这些算是完整通了 A 面，接下来还有 B 面要通关。等玩家好不容易通关了 B 面，居然还有一个 C 面可以通关，且 C 面开启后，每个章节（ABC 面的所有章节）都可以收集一个黄草莓，只有一命不死通过一章后才能获得一个黄草莓。（这也太难了吧。。。）

玩家还能在游戏中发现一台小游戏机，游戏机中搭载了一个看起来像是 Celeste 最初 demo 的游戏，在这个小游戏中也是一关一关的进行，但是却不能保存进度，所以我现在也不知道这个小游戏打完会有什么东西。（太菜了根本打不通啊。。。）

写到这里其实就能发现 Celeste 的内容确实很丰富，可玩性很强，全收集的难度较大，也算是易于上手难于精通的游戏了吧。

在游玩 Celeste 的过程中，不但会考验到玩家的操作能力，很多时候也会考验到玩家对于通关路线的规划能力。前期关卡有些地方有多种走法，规划好路线会降低操作的难度。到了 B 面或者 C 面后，许多关卡大多只有一种走法，将操作难度提升到了最大，不过这种超难关卡的唯一走法大多包含一些匪夷所思的操作，因此这种关卡也需要经过不断试错才能探索出正确的路线。

虽然这游戏后期确实很难，但是通过一个关卡后获得的巨大成就感完全可以盖过死亡时的挫败感。如果玩家有条件录下自己的通关过程，还能达成一次通关，循环装逼的效果。

手感

平台跳跃类游戏能否成功的一个关键因素就是手感，手感不好的操作势必会加重玩家的挫败感。

Celeste 的手感十分完美，鉴于游戏中的震动反馈很及时又很有节奏感，建议玩家们都开启 100% 的震动游玩游戏。

Celeste 中的动作反馈都很直接，这能给玩家一种尽在掌控的感觉。值得一提的是突进机制，这个机制不但增加了游戏的可操作性，同时也使得游戏观赏性更强。

提到手感就不得不提一下手柄对游戏操作的影响，由于我是用 ns 的掌机模式进行游戏的，所以一直用的都是 joycon 手柄。joycon 手柄的缺陷就是方向键键程（大概是这个意思）太短了，因为这个时常会打出误操作，比如向右会变成向右下等等。建议大家还是用好一点的手柄来玩这款游戏。

同时这款游戏实在太适合用 ns 掌机模式玩了（其实我觉得所有独立游戏都很适合），强力安利一下。
